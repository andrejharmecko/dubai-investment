# Dubai Investment Dashboard

Interactive financial dashboard for tracking two Dubai real estate investments:
- **Palm Beach Towers** (50% share) — Palm Jumeirah, 1BR, 22nd floor
- **Avenew 888** — Dubai South, 2BR, 800sqft

## Features
- Live-updating calculations as you change any assumption
- Mortgage analysis with PMT calculations
- 10-year ROI projections with property appreciation
- Payment timeline tracking
- Cumulative cash flow visualization
- Debt overview with repayment calculator

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

```bash
npx vercel
```

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- Recharts
